# UnbrokenSolutions Website

A simple reference website built with **React JS** and **CSS** for a logistics and printing solutions business.  
This was my first freelance project, created for a friend who needed an online presence to showcase services.

---

## 🚀 Tech Stack
- [React JS](ca://s?q=React_JS_overview) — component-based frontend library
- [CSS](ca://s?q=CSS_for_styling) — styling and layout

---

## ✨ Features
- Responsive layout for desktop and mobile
- Navigation bar with multiple pages
- Service information and contact details
- Clean, minimal design tailored to business branding
